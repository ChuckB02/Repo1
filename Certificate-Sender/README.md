# Certificate Sender

Send Bulk Certificates Automatically via mail.

- send bulk certificaates
- add names & other custom details on Certificate
- easy to use

## Setup instructions

- Add all details in `.env` file
- Setup SMTP in Gmail
- Add your certificate template & set position of Name Field
- Add font
