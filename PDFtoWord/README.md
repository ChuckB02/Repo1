## Description:

- This script converts a pdf file to a word document
- Can be used if a pdf has to be edited

## Libraries used :
- pdf2docx

## Steps to run the code:

If you have cloned this repository , 

```
cd PDFtoWord
pip install pdf2docx
python main.py

```
